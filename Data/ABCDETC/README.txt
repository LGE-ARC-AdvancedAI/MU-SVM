1. Please reach out to the authors of the paper " Weston, Jason, et al. "Inference with the universum." Proceedings of the 23rd international conference on Machine learning. ACM, 2006." for the original data "final.svl". This data is in LIBSVM format. 

2. Use the interface provided in this package as: - 
[data]= libsvm2mat('final.svl')
This will convert the data in the needed .mat format.

3. Save the data in the MATLAB path using:- save abcddata
